# Sales Data Analysis with Python

## Project Overview
This project performs sales data analysis using Python to uncover business insights from a retail dataset. The analysis focuses on sales trends, customer behavior, product performance, and regional sales distribution.

The project uses Python libraries such as pandas, matplotlib, and seaborn to clean, analyze, and visualize the data.

---

## Dataset
The dataset used in this project is the Superstore sales dataset which contains information about orders, customers, products, and sales performance.

Dataset includes the following columns:

- Order ID
- Order Date
- Ship Date
- Customer Name
- Segment
- Country
- City
- Region
- Category
- Sub-Category
- Sales
- Quantity
- Profit

## Business Insights

Some insights from the analysis include:

- Technology category generates the highest sales.
- The West region contributes the most to overall revenue.
- Certain products generate high sales but lower profit.
- Monthly sales trends show seasonal demand patterns.
